# /community-post — Generate YouTube Community Post

Create engaging community posts (polls, updates, behind-scenes) to boost subscriber engagement.

## Arguments
- `/community-post` — Auto-generate from today's work
- `/community-post poll "Question"` — Create a poll
- `/community-post update` — Progress update

## Steps

1. **Generate post** based on type:
   - Poll: Binary choice about architecture decisions
   - Update: Progress milestone with numbers
   - Behind-scenes: Terminal screenshot with context
2. **Post** via YouTube Data API community endpoint
