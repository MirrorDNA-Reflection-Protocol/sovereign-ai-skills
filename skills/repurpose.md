# /repurpose — Repurpose Content Across Formats

Take one piece of content and multiply it into 7+ formats.

## Arguments
- `/repurpose <video_id>` — Repurpose a YouTube video
- `/repurpose --blog "post-slug"` — Repurpose a blog post

## Steps

From ONE video/post, generate:
1. YouTube Long-form (original)
2. 3-5 YouTube Shorts (key moments)
3. Bluesky thread (10 posts)
4. Blog post (beacon)
5. dev.to article
6. Quote cards (3-5 images)
7. Newsletter segment
8. GitHub README update

Each format optimized for its platform's algorithm.
