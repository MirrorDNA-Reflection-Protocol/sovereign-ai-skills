# /schema — Add Structured Data / Schema.org

Add JSON-LD structured data to activemirror.ai for rich search results.

## Steps

1. **Generate** schema for:
   - Organization (Active Mirror)
   - Person (Paul Desai)
   - SoftwareApplication (MirrorDNA)
   - VideoObject (for each YouTube embed)
   - Article (for each blog post)
2. **Inject** into site HTML head
3. **Validate** with Google Rich Results Test
