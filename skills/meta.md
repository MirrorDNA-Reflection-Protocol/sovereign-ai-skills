# /meta — Optimize Page Meta Tags

Audit and optimize meta tags (title, description, OG) across all web properties.

## Steps

1. **Crawl** activemirror.ai pages
2. **Check** each page for:
   - Title tag (50-60 chars, keywords)
   - Meta description (150-160 chars)
   - OG image, OG title, OG description
   - Canonical URL
3. **Generate** optimized replacements
4. **Apply** changes to source repos
