# /milestone — Celebrate & Share Milestones

Auto-detect and celebrate milestones (subscriber count, days building, features shipped, etc.).

## Steps

1. **Check milestones**:
   - YouTube subscriber count
   - Days since project start (from FACTS.md)
   - Total features in SHIPLOG
   - Total vault notes
   - GitHub stars across repos
2. **If milestone hit**: Generate celebration post
3. **Cross-post** to all platforms with custom graphic
