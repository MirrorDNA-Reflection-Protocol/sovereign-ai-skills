# /evergreen — Create Evergreen Content

Generate content that stays relevant and accumulates views over time (how-to, setup guides, comparisons).

## Arguments
- `/evergreen` — Generate top 10 evergreen ideas from vault
- `/evergreen "topic"` — Create evergreen piece on topic

## Steps

1. **Identify** evergreen topics from vault knowledge:
   - "How to run AI locally on Mac Mini M4"
   - "Complete guide to sovereign AI stack"
   - "Ollama vs OpenAI: Real cost comparison"
2. **Generate** SEO-optimized outline
3. **Create** blog post + video script
4. **Optimize** for search (long-tail keywords)
