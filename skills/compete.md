# /compete — Analyze Competitor YouTube Channels

Analyze similar channels (Network Chuck, Fireship, ThePrimeagen, etc.) for what's working.

## Arguments
- `/compete` — Quick scan of top 5 competitors
- `/compete <channel_id>` — Deep dive on specific channel

## Steps

1. **Fetch** recent videos from competitor channels via YouTube Data API
2. **Analyze** titles, descriptions, tags, view counts, engagement rates
3. **Identify patterns**: what titles get views, what tags work, posting frequency
4. **Generate** actionable recommendations for Paul's channel
