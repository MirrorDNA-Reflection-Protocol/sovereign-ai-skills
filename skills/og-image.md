# /og-image — Generate Open Graph Images

Generate OG images for web pages and blog posts (shown when shared on social).

## Arguments
- `/og-image "Title"` — Generate OG image

## Steps

1. **Create** 1200x630 image via ffmpeg:
   - Dark gradient background
   - Title text (large, bold)
   - Subtitle (activemirror.ai branding)
   - Avatar/logo in corner
2. **Output** as PNG ready for deployment
