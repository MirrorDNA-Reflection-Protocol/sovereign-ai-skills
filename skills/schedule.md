# /schedule — Optimal Content Schedule

Calculate and set the optimal posting schedule based on audience analytics and platform algorithms.

## Steps

1. **Analyze** YouTube Analytics for best posting times
2. **Check** Bluesky engagement patterns
3. **Generate** weekly content calendar:
   - Daily: Build log video (11pm IST)
   - Daily: 3 Shorts from recordings
   - Daily: Bluesky thread
   - Weekly: Recap video (Sunday)
   - Weekly: Newsletter (Monday)
   - Bi-weekly: Tutorial
   - Monthly: Comparison/challenge video
4. **Update** LaunchAgent schedules to match
