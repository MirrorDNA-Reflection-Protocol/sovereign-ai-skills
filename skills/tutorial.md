# /tutorial — Extract Tutorial from Build Log

Turn a build session into a step-by-step tutorial (video + blog + code).

## Arguments
- `/tutorial "topic"` — Generate tutorial for topic

## Steps

1. **Identify** the teachable moments from today's recordings
2. **Generate** step-by-step instructions via Ollama
3. **Extract** relevant code snippets from repos
4. **Compile** into tutorial format:
   - Blog post (beacon + dev.to)
   - Video with annotations
   - Code examples (GitHub gist)
