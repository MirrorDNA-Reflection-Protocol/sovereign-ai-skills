# /storytell — Narrative Content Generator

Transform technical build logs into compelling stories using narrative structure.

## Arguments
- `/storytell` — Today's story
- `/storytell "arc"` — Specific narrative arc

## Steps

1. **Read** today's SHIPLOG + session reports
2. **Apply** story structure:
   - Setup: What was the goal?
   - Conflict: What went wrong?
   - Resolution: How was it solved?
   - Lesson: What was learned?
3. **Generate** in multiple formats:
   - Video script (narrated build log)
   - Blog post (beacon)
   - Social thread (Bluesky)
