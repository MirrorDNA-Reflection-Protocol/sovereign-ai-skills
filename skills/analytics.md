# /analytics — Cross-Platform Analytics

Pull and display analytics from all content platforms.

## Steps

1. **YouTube** (via Data API):
   - Subscribers, total views, watch hours
   - Last 5 videos: views, CTR, AVD, likes
2. **Bluesky**: Follower count, recent post engagement
3. **GitHub**: Stars, forks, profile views
4. **Beacon**: Post count, RSS subscriber estimate
5. **Generate** trend summary and recommendations
