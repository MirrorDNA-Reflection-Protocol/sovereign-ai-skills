# /recap — Weekly/Monthly Content Recap

Generate a recap video + thread + blog post summarizing a week or month of building.

## Arguments
- `/recap week` — This week's recap
- `/recap month` — This month's recap

## Steps

1. **Aggregate** SHIPLOG entries for the period
2. **Generate narrative** via Ollama (story arc: challenge → build → ship)
3. **Compile video** from week's best recordings
4. **Write blog post** for beacon
5. **Generate social thread**
6. **Cross-post** to all platforms
