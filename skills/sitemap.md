# /sitemap — Regenerate Sitemaps

Regenerate and submit sitemaps for all Active Mirror domains.

## Steps

1. **Regenerate** sitemap for activemirror.ai
2. **Regenerate** sitemap for beacon.activemirror.ai
3. **Regenerate** sitemap for docs.activemirror.ai
4. **Submit** to Google Search Console (if API key available)
5. **Verify** all pages are indexed
