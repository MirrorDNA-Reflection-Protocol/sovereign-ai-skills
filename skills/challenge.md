# /challenge — Create Coding Challenge Content

Generate a coding challenge video: "Can I build X in Y minutes?" format.

## Arguments
- `/challenge "Build X"` — Generate challenge concept

## Steps

1. **Define challenge** scope (buildable in 1 recording session)
2. **Generate** title using viral formula: "I Built [X] in [Time] — No Cloud, No Team"
3. **Create** thumbnail template
4. **Draft** video structure:
   - Hook (10 sec): Show the end result
   - Setup (30 sec): What we're building and why
   - Build (main): Real-time with commentary markers
   - Result (30 sec): Demo the working thing
