# /viral-audit — Viral Readiness Audit

Score each video and post against viral readiness criteria.

## Steps

1. **Score each recent video** (1-10) on:
   - Title: Hook strength, keyword density, length
   - Thumbnail: Contrast, text readability, curiosity gap
   - Description: First 150 chars, keywords, links, hashtags
   - Tags: Relevance, count, long-tail vs broad
   - Engagement: Like ratio, comment rate, share rate
2. **Identify** lowest-scoring videos
3. **Auto-fix** what can be fixed (titles, descriptions, tags)
4. **Report** what needs manual fixes (thumbnails, content)
