# /brand — Brand Consistency Checker

Audit all platforms for consistent branding (name, bio, links, colors, messaging).

## Steps

1. **Check** YouTube channel: name, description, links, banner
2. **Check** Bluesky profile: handle, bio, avatar
3. **Check** GitHub profile: bio, pinned repos, README
4. **Check** activemirror.ai: meta tags, OG images
5. **Check** beacon: branding, links
6. **Flag** inconsistencies and auto-fix what's possible
