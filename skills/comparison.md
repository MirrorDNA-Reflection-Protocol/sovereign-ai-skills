# /comparison — Create Comparison Content

Generate "X vs Y" comparison content — the highest CTR format on YouTube.

## Arguments
- `/comparison "Local AI vs Cloud AI"` — Generate comparison

## Steps

1. **Generate** comparison framework via Ollama:
   - Cost comparison (with real numbers from FACTS.md)
   - Performance comparison
   - Privacy/sovereignty comparison
   - Setup complexity comparison
2. **Create** side-by-side visuals
3. **Write** script with balanced but opinionated take
4. **Output** as video script + blog post + social thread
