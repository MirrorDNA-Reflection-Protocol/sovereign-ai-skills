# /headline — Generate Headlines & Titles

Power-generate 20 headline variants for any content using proven viral formulas.

## Arguments
- `/headline "topic"` — Generate headlines for topic

## Steps

1. **Apply 5 formulas** to generate 20 variants:
   - How-to: "How I Built X Without Y"
   - List: "7 Things I Learned Building X"
   - Contrarian: "Stop Using Cloud AI. Here's Why."
   - Story: "I Quit My Job to Build AI. 10 Months Later..."
   - Challenge: "Building X in 24 Hours — No Cloud Allowed"
2. **Score** each on: clarity, curiosity, keywords, length
3. **Output** top 5 with scores
