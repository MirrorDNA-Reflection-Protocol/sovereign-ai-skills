# /series — Plan Content Series

Generate a multi-part content series plan optimized for YouTube algorithm (series get 3x more suggested views).

## Arguments
- `/series "topic"` — Plan a series around topic
- `/series --episodes 10` — Number of episodes

## Steps

1. **Generate series concept** via Ollama:
   - Series title (branded, searchable)
   - Episode titles (numbered, progressive)
   - Thumbnail template (consistent branding)
   - Upload schedule (optimal for algorithm)
2. **Create playlist** via YouTube API
3. **Generate episode outlines** with hooks
4. **Output** full series plan as vault note
